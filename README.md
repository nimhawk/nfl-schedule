### nfl-schedule

Playground App for displaying (Patriots) 2015 NFL schedule/results

## Technologies
- Node
- es6 via Babel
- Express
- React/Flux
- Bootstrap/Sass
- Webpack
- Mocha/Should/Sinon/Proxyquire/Istanbul
